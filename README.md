# Foundry

**Research-to-Projects Workspace**

Feed Foundry a YouTube video, paper, repo, or article. It analyzes the content, extracts the distinct buildable projects described inside, and organizes them into structured workspaces where you and an AI agent continue building.

> Open by default. Reliable by choice. Dangerous only by consent.

## Status

🚧 **Under active development** — not yet ready for general use.

## Quick Start

```bash
# Prerequisites: Python 3.11+, Node.js 18+, git
git clone https://github.com/Patvscode/foundry.git
cd foundry
make dev-setup
make dev
# Open http://localhost:8120
```
